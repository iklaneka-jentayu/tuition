# ExcelLearn Tuition Center - Complete Web Application

A comprehensive online and offline tuition center management system with dual language support (English & Malay), landing page, admin dashboard, and Google Apps Script backend.

## Features

### 🎯 Dual Language Support
- English and Bahasa Malaysia language options
- Language selector dropdown in header
- Persistent language preference (saved in localStorage)
- SEO-friendly URLs with language support

### 🌐 Landing Page
- Responsive design for desktop and mobile
- SEO-optimized copywriting in both languages
- Enrollment form with validation
- ToyyibPay integration for payments
- Contact form and newsletter subscription

### 🔐 Admin Dashboard
- Secure login system for admin and members
- CRUD operations for student management
- Enrollment and payment tracking
- Data export functionality
- Real-time statistics and reporting

### 🔧 Backend Integration
- Google Sheets as database
- Google Apps Script for CRUD operations
- No CORS issues with proper setup
- ToyyibPay payment gateway integration
