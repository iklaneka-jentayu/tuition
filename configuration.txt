
## Configuration

### Google Apps Script Web App
1. Deploy as Web App with these settings:
   - Execute as: "Me"
   - Who has access: "Anyone" (or "Anyone with Google account" for production)
2. Note the Web App URL

### ToyyibPay Configuration
Update these values in `Code.gs`:
- `toyibpayApiKey`
- `toyibpayUserSecretKey`
- `categoryCode`
- Callback URLs

## Security Notes

1. **For Production Use**:
   - Implement proper authentication in Google Apps Script
   - Use HTTPS for all connections
   - Store API keys securely (not in client-side code)
   - Implement CSRF protection

2. **Data Security**:
   - Regular backups of Google Sheets data
   - Access control for admin dashboard
   - Input validation on all forms

## Troubleshooting

### Common Issues

1. **Google Apps Script CORS Errors**
   - Ensure Web App is deployed correctly
   - Check permissions are set to "Anyone"
   - Verify the URL is correct in frontend code

2. **ToyyibPay Integration Issues**
   - Verify API keys are correct
   - Check callback URLs are properly configured
   - Ensure SSL certificates are valid

3. **Data Not Saving to Google Sheets**
   - Check Google Apps Script execution logs
   - Verify spreadsheet permissions
   - Test the Web App endpoint directly
